Real-Time Quantitative Trading Analytics Dashboard
Description

This application is a real-time quantitative trading analytics dashboard built for monitoring
Binance Futures markets. It streams live tick-level data, performs pair-trading statistical
analysis, and presents the results through interactive visualizations.

Key Features

Live data ingestion from Binance Futures

Real-time computation of pair-trading metrics

Interactive charts and analytics powered by Plotly

Lightweight and responsive UI built with Streamlit

Installation & Setup

Install the required dependencies:

pip install streamlit websocket-client pandas numpy plotly statsmodels


Launch the application:

streamlit run app.py

Usage

Once the app is running, open the provided local URL in your browser to view live market data,
pair-trading statistics, and interactive visual insights in real time.
